CLI Command Flashcards

sudo

Acronym for superuser do or subtitute user do. Sudo allows a user with proper permission to execute a command as another user.

pwd

Look at directory that you are in.

cd

Change directories/ go to home directory

ls

list of directories.

mv

Move a file (You can also rename it)

mkdir

Make a directory.

rmdir

Remove a directory

rm

Remove

touch

Create a file (Puts a date to it)

find

The find command is used to search and locate the list of files and directories based on conditions you specify for files that match the arguments.

grep

Acronym for Global Regular Expression Print. It is used to search for a string of characters in a specified file.

df

Tells the user the difference between two files.

du

Stands for Disk Usage. The commands is used to check the information of disk usage of files and directories on a machine.

head

Inquire what is in front of a file.

tail

Inquire what is in the end of a file.

diff

Stands for difference. The command is used to display the differences in the files by comparing the files line by line.
